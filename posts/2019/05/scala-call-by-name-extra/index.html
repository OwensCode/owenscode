Scala call-by-name Extra

May 8, 2019

A short addendum to my last post about the call by name feature in Scala.

Something interesting about the loop function that I used is that the Java equivalent of that function would die with a StackOverflowError, but the Scala version doesn’t because it’s tail recursive and Scala is able to reuse the stack frame in each recursive call, resulting in an infinite loop instead of an error.

We can try it out by changing the loop function to not be tail recursive by incrementing the expression result by 1.

scala> def loop(): Int = loop() + 1
loop: ()Int

scala> loop()
java.lang.StackOverflowError
  at .loop(<console>:11)
  at .loop(<console>:11)
  ...
